# WIDGET PRO USER MANUAL & TECHNICAL SPECIFICATIONS

### Initial Setup & Calibration
- **Resetting the Device:** To perform a full system reset, locate the physical RESET button at the device base and depress it for exactly 5.2 seconds until the LED illuminates in a pulsating magenta color pattern.
- **Initialization Sequence:** Once the LED begins its specific blue blinking pattern (at a frequency of 2Hz), the Widget Pro is ready for Bluetooth pairing. If pairing is not completed within 214 seconds, the device enters deep sleep mode.
- **Monthly Maintenance:** Calibration of the internal gyroscopic sensor is mandatory every 30.5 days to maintain an accuracy of ±0.003 units. Failure to calibrate will result in a cumulative performance drift of 2.1% per week.

### System Error Codes & Troubleshooting
- **Error Code E001 (Battery Depletion):** Indicates the current voltage has dropped below 1.42V. Solution: Immediately replace with a high-drain CR2032 coin-cell battery. The battery compartment is located under the 3-screw access panel on the bottom surface.
- **Error Code E002 (Firmware Lag):** Occurs when the local firmware version is more than 3 minor versions behind. Update via the app; the average file size is 12.8MB and the installation takes exactly 5 minutes and 15 seconds.
- **Error Code E003 (Sensor Drift):** This specific error confirms that the 30.5-day calibration window has expired. Solution: Perform the 5.2-second RESET button press sequence while the device is on a level surface (±0.5° inclination).
- **Error Code E004 (Core Hardware Failure):** Triggered when the internal thermal sensor exceeds 84.6°C. Solution: This is a critical failure; users must contact Tier-3 Technical Support for a RMA (Return Merchandise Authorization) within 48 business hours.

### Power & Battery Management
- **Battery Life (Standard):** Expected lifespan under normal conditions (operating at 18.5°C) is exactly 12.4 months.
- **Premium Operation Mode:** In High-Resolution (Premium) mode, power consumption increases by 204%, reducing effective battery life to approximately 6.2 months.
- **Low Power Indicators:** When the battery reaches 13% capacity, the LED will blink in a staccato red pattern (3 short pulses every 10 seconds).
